# StealthForward 开发路线图 (Roadmap)

## 🎯 当前状态: v3.3.3 (极速加速版)
- [x] **极限首开优化**: 嗅探超时压至 100ms，实现 YouTube 首屏“秒开”。
- [x] **淡黄书页高级 UI**: 实现护眼且具备高级审美的前端界面。
- [x] **断流深度加固**: RLimit 自动配置、平滑加载、65535 并发连接支持。
- [x] **稳定性补丁**: 彻底修复系统设置页白屏与数据加载竞态问题。

---

## 🚀 未来计划: Phase 5 - 自动化与资源池化

### 1. 云服务资源池 (Credential Pool) - [启动中]
- **功能描述**: 支持存储 10~20 个 AWS 账号，实现“无感知”轮转。
- **核心逻辑**:
  - **动态轮询**: 创建实例或换 IP 时，自动从池子里选一个“最闲”或“健康”的 Key。
  - **自动故障转移**: 遇到限额（Limit Exceeded）自动跳过，尝试下一个 Key。

### 2. 远程零接触初始化 (Zero-Touch Provisioning) - [启动中]
- **功能描述**: 主控开机后，自动完成环境配置。
- **核心逻辑**:
  - **SSH 自动执行**: 机器就绪后，主控自动 SSH 登录。
  - **环境一键刷写**: 自动开启 BBR、优化内核参数、设置最大句柄。
  - **Agent 自动对接**: 自动下载二进制并启动，全程无需人工干预。

### 3. 流量持久化存档 (Traffic Persistence) - [构思中]
- **功能描述**: 增加 Agent 本地缓存数据库，防止重启丢量。

---

## 🛠️ 后期规划: Phase 6 - 智能化与安全

- [ ] **异常行为预警**: 基于流量模型的“用户盗用”或“连接异常”自动弹窗。
- [ ] **多层级管理员功能**: 支持分级代理管理自己的账户组。
- [ ] **全球节点延迟测绘图**: 在 UI 上展示全球各落地点的实时延迟热力图。
