# StealthForward (隐形转发面板)

## 项目简介
StealthForward 是一款专为**海外入口机**设计的、具备极高隐身能力的协议化转发面板。
它通过将传统的端口转发“协议隧道化”，利用 `VLESS + XTLS-Vision` 或 `Reality` 等先进协议作为入口，彻底解决海外机直接做转发入口时 IP 易被墙的问题。

## 核心设计
- **入口隐身**：模仿正常 HTTPS 流量，躲避 DPI 检测。
- **逻辑转发**：基于用户标识（UID/Email）进行后程分流。
- **多级落地**：支持灵活配置不同的出口机器。
